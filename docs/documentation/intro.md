# Introduction

Welcome to the official technical documentation for HyperEdu, a modern web application built with Laravel (PHP) backend and React (TypeScript) frontend.

## Overview

HyperEdu is designed to provide an educational platform with robust backend services and an intuitive user interface. This documentation covers all aspects of the application's architecture, development, deployment, and maintenance.

### Key Technologies

- **Backend**: Laravel 10+ (PHP 8.1+)
- **Frontend**: React 18+ with TypeScript
- **Database**: MySQL/PostgreSQL
- **API**: RESTful API with JWT authentication
- **Deployment**: Docker containers with CI/CD pipeline
